# 🦆 かもん！カモねぎくん

ようこそ！ここは **TomoDesign Works** が手がける
地域文化発信をするホームページ **かもん！カモねぎくん** の公式リポジトリです。

---

## 🌱 プロジェクト概要
- **目的**: **かもん！カモねぎまつり**の情報やキャラクターの**カモねぎくん**に関連する画像データ等の公開  
- **キャラクター**: カモねぎくん（カモ＋ネギ＋鍋の組み合わせから生まれた、親しみやすい存在）  
- **技術基盤**: 完全手打ちによるHTML&CSSをGitHub Pagesで公開
- **理念**: 公共性・再現性・意味論的構造を重視した情報発信

---

## 🐤 カモねぎくんとは？
- カモねぎまつり第1回開催の1996年に作成したキャラクター〈おまつりマスコット〉
- ロゴやイラストは SVG で管理し、誰でも再利用可能な形で公開
